# ProjectedPSNR
Projected PSNR metric (P-PSNR).

The projected_PSNR_cube function is a Matlab/Octave function which provides the Projected PSNR metric (P-PSNR) as proposed on ISO/IEC JTC1/SC29/WG1 input document M78030.
